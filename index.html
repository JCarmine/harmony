A FULLY RESPONSIVE SIGN-UP FORM.

This template was designed by BBK Worldwide, and developed by Jesse Carmine. It provides a place where parents can go to enroll their children in a clinical research study. Please note that the original content has been removed per the client’s request, and the internal links have been disabled. The site features a background image that uses jQuery to scale its width and height. This allows the background to maintain its proportions and properly align itself with the page’s contents. Additional site features include a fully responsive design, mobile menu, and custom form.
